# Fahrenheit to Celsius Converter (Python)

About the Project
This is a simple **Fahrenheit to Celsius converter** written in **Python**.  
I created this project as part of my learning process while studying Python.

## 🧮 How It Works
The program:
1. Asks the user to enter a temperature in **Fahrenheit or Celsius**
2. Converts the value to **Celsius/Fahrenheit**
3. Prints the result to the console
